# NeuroScribe Project Context

NeuroScribe is an AI-powered clinical report intelligence platform.

Current Stack:

* FastAPI
* Python
* FAISS
* Sentence Transformers
* Supabase (planned migration)
* Next.js frontend
* RAG architecture

Current Features:

* PDF processing
* Text preprocessing
* Chunking
* Embedding generation
* Persistent FAISS vector database
* Medical query expansion
* Hybrid retrieval scoring
* Clinical entity extraction
* Hallucination prevention foundations
* Deterministic medical QA logic

Important Rules:

* Never rewrite architecture.
* Always make minimal changes.
* Preserve existing APIs.
* Show file modifications before code generation.
* Explain reasoning before implementation.
* Prefer new service files over modifying working modules.

Current Goal:
Continue development feature-by-feature.
